Mac Agent Gateway

A local macOS HTTP API gateway for AI agents to access Apple Reminders and Messages. Provides REST API endpoints for interacting with native macOS services via remindctl and imsg.

Endpoints:

/v1/reminders - Manage Apple Reminders
/v1/messages - Send iMessages
